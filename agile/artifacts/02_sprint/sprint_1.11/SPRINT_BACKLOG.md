## 🎯 Sprint Goal

### What are we building?
We are polishing the refreshed BetterPrompt design so it feels more consistent across the detailed assessor experience. This sprint focuses on result cards, mobile spacing, and the floating tutor panel.

### Expected Outcome (What you will see)
By the end of this sprint, you will be able to:
1. See cleaner, more consistent result cards in the assessor flow.
2. Use the assessor more comfortably on smaller screens.
3. Experience a tutor panel that visually matches the refreshed BetterPrompt shell more closely.

### Acceptance Criteria
- [x] Result cards and response surfaces use the refreshed design language consistently.
- [x] Main assessor spacing and button behavior feel better on mobile.
- [x] The BetterPrompt Tutor panel matches the global design system more closely.
