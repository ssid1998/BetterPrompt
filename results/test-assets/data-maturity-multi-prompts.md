# Data Maturity Assessment Prompt Pack

## Suggested User Intent
I want to build a data maturity assessment tool for companies.

## Prompt 1
Act as a senior data strategy consultant. Define the main dimensions of a data maturity assessment for a growing business.

## Prompt 2
For each dimension, create five practical assessment questions that can be answered by business stakeholders.

## Prompt 3
Create a scoring rubric from 1 to 5 for each dimension, where 1 means ad hoc and 5 means optimized.

## Prompt 4
Rewrite the entire assessment into a format that could be used in a web application questionnaire.

## Prompt 5
Suggest next-step recommendations for organizations that score low, medium, and high.
