# Data Maturity Assessment Tool Demo Prompts

## 1. Vague Prompt

**Goal:** Build a data maturity assessment tool.

**Prompt:**

Build me a data maturity assessment tool for a company. Make it practical and useful for business leaders. Include the main things that should be assessed and a way to score it.

---

## 2. Clear Prompt

**Goal:** Build a data maturity assessment tool.

**Prompt:**

Create a data maturity assessment tool for a mid-sized organization. The tool should help leadership teams understand how mature they are across areas like data governance, data quality, analytics, technology, and team capabilities. Include assessment categories, a scoring scale, a short description of each maturity level, and a summary of recommended next steps based on the score.

---

## 3. Precise Prompt

**Goal:** Build a data maturity assessment tool.

**Prompt:**

You are a senior data strategy consultant. Design a business-ready Data Maturity Assessment Tool for a mid-sized organization.

Context:
- The audience is CIOs, data leaders, transformation leads, and business executives.
- The tool should help assess current-state maturity and identify improvement priorities.
- The assessment should cover governance, data quality, platform/infrastructure, reporting and analytics, operating model, people and skills, and AI readiness.

Requirements:
- Define 7 maturity dimensions.
- Use a 1-5 scoring scale for each dimension.
- Provide a clear definition for each maturity level.
- Include 4-6 example assessment questions per dimension.
- Add a method for calculating an overall maturity score.
- Include guidance on how to interpret low, medium, and high scores.
- Recommend next-step actions for each score range.

Output format:
- Start with a short overview.
- Then provide a table of dimensions and scoring criteria.
- Then list example assessment questions by dimension.
- End with a score interpretation and recommended actions section.

Constraints:
- Keep the language executive-friendly and workshop-ready.
- Make the framework specific enough to use in a client assessment session.
- Avoid generic advice.
