# Anthropic Structured Prompt Test

## Suggested User Intent
I want Claude to help me create a data maturity assessment framework.

```xml
<context>
I am building a university demo for a data maturity assessment tool for companies.
The audience is business leaders who are not deeply technical.
</context>

<instructions>
Act as an expert in data strategy and assessment design.
Create:
1. Core maturity dimensions
2. Sample diagnostic questions
3. A simple scoring rubric
4. Recommendations by maturity level
Return the answer in a clean structured format.
</instructions>
```

## Extra Variation
Also compare this against a weaker prompt that simply says: "Tell me how to make a data maturity assessment tool."
