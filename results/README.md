# BetterPrompt (Sprint 1.1 MVP)

This is the Greenfield Next.js application for BetterPrompt. 

## How to Run

We have included helper scripts to make running the application easy and conflict-free:

### Mac / Linux
```bash
./start.sh
```
*(To stop the server manually, you can run `./stop.sh`)*

### Windows
```cmd
start.bat
```
*(To stop the server manually, you can run `stop.bat`)*

## Features Currently Implemented
- Next.js & Tailwind CSS foundation.
- Responsive UI for inputting prompts.
- User Intent (Goal) capture.
- Model Provider selection (OpenAI, Anthropic, Google).
- Basic input validation (minimum 10 words).

*(Assessment engine and backend logic coming in the next sprint!)*
